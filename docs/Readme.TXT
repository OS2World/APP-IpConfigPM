IpConfigPM version 1.4
============================
2013-01-15
-----------
Dusan Golubic relased this project as Open Source under the GNU GPL V3 license. 

IPConfigPM is a graphical interface to change IP address for network addapters Domain name and DNS servers using operating system eComstation and OS/2 Warp.

Development Notes
-----------------
This software was developed using Dr.Dialog for OS/2.

Warning:
---------
This code is likely to be completely untested. USE AT YOUR OWN RISK!
It is likely to contain bugs, have some functions broken and contain 
partially implemented features.

Bug reports, feature requests or any other feedback may be sent to
the author, Dusan Golubic <dusan(at)lumic(dot)si>

I need feed back from one who is not using English version of eComstation!
